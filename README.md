My personal website created in Windows95 style containing:
- my projects (portfolio)
- my programming skills
- a short description of me
- my socials and contact
- and other things to spice up the use of the website

The site is made using the [React library](https://react.dev/) and [React95 component library](https://github.com/React95/React95)

The site uses [Webamp](https://github.com/captbaritone/webamp) to enhance the use of the site (playing music), with the theme [Virtual_Reality.wsz](https://skins.webamp.org/skin/5a214ec76a65200aa4c188e5e444b7de/Virtual_Reality.wsz/)
